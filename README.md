# Site for football competition 
⚽ Memorial Football Tournament Website
📌 Overview

This project is a website created as an informational and registration platform for a football tournament held in memory of our friend who tragically passed away in a car accident.

He truly loved football, and this tournament is our way of honoring his memory and bringing people together through the sport he cared about so much.

🎯 Purpose

The website was designed to:

Provide information about the tournament

Share details about the event and its purpose

Allow participants to register  through Google Forms

🌐 Features

Informational landing page about the tournament

Clear presentation of event details

Integrated registration via Google Forms

Simple and user-friendly interface

❤️ Motivation

This project is more than just a website — it’s a tribute.
It reflects our desire to remember a friend, celebrate his passion for football, and create something meaningful in his honor.
